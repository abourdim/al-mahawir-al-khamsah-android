# 🔥 The Five Axes of the Quran — المحاور الخمسة للقرآن الكريم

**بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ**

Interactive trilingual (AR/EN/FR) educational web app based on the book **المحاور الخمسة للقرآن الكريم (The Five Axes of the Quran)** by Sheikh Mohammed al-Ghazali (1917-1996).

## Features

- 20 cards on the 5 Quranic axes (Tawhid, Creation, Prophets, Afterlife, Ethics/Law), 8 reflections, daily habit tracker, self-quiz
- Trilingual: Arabic, English, French with RTL/LTR support
- 3 unique themes, sound effects, responsive, PWA-ready

## Tech

Pure HTML + CSS + JS. No frameworks. Just open `index.html`.

## Disclaimer

I am not a scholar. This is a humble effort by a Muslim parent to share Sheikh al-Ghazali's wisdom interactively.
